# 主窗口

CTRL-O : 打开新文件 CTRL-S : 导出PKM CTRL-E : 导出SAV CTRL-B : 导出 BAK CTRL-Q : 退出

CTRL-D : 打开PKM数据库 CTRL-F : 打开文件夹列表 CTRL-G : 打开神秘礼物数据库 CTRL-N : 打开相遇数据库 CTRL-M : 打开批处理编辑器 CTRL-R : 打开框报告 CTRL-P : 打开关于PKHeX CTRL-T : 导入Showdown代码 CTRL-SHIFT-T : 导出Showdown代码 CTRL-SHIFT-S : 打开设置

# CTRL + 左击

-种类标签 : 从剪贴板导入Showdown/Smogon设置。 -昵称/名称输入框 : 打开游戏中的特殊字符。 -单个IV输入框 : 设置IV为31(如果已经是31则为30)。 -单个AV输入框 : 将AV设置为max(如果已经是max则设置为0)。 -单个EV输入框 : 将EV设置为“最大值”。 -随机个体值 : 将所有IV设置为最大值。 -随机觉醒值 : 将所有AV设置为最大值。 -随机努力值 : 将种族值最高的两项的努力值设为252，第三项为6，存在相同数值时随机一项。 -属性标签 : 选择一个无加成的薄荷性格 -PP提升标签 : 将所有最大PP值设置为0。 -亲密度标签 : “友谊”设置为“0”。 -等级输入框 : 设置“级别”为“100”。 -闪光按钮 : 宝可梦闪光(Xor=1) -原始性格标签 : 追随薄荷修正 -薄荷修正标签 : 追随原始性格 -身高输入框 : 随机生成身高 -体重输入框 : 随机生成体重 -大小输入框 : 随机生成大小

# Alt +点击…

-预览精灵 : 从剪贴板上的QR url加载。 -种类标签 : 导出Showdown/Smogon设置为剪贴板。 -随机个体值 : 所有IV均为0。 -随机觉醒值 : 将所有AV设置为0。 -随机努力值 : 将所有EV设置为0。 -单个IV输入框 : 设置IV为0。 -单个AV输入框 : 将AV设置为0。 -单个EV输入框 : 将EV设置为0。 -闪光按钮 : 选择闪光时设置SID而不是PID -属性标签 : 选择一个对该属性削减的薄荷性格

# Shift +点击…

-预览精灵 : 生成二维码。 -单个IV输入框 : 极限训练(Gen 7开始有效) -闪光按钮 : 使其方框闪(Xor=0) -招式标签区域 : 随机设置招式。 -属性标签 : 选择一个对该属性加强的薄荷性格

拖动以保存加密(ekx)时按住Control。

# 点击…

-OT标签 : 将保存文件详情设置为相关属性。 -等级 / 相遇标签 : 设置为建议的遭遇信息。 -招式标签区域 : 设置为建议的合法招式。 -回忆招式标签区域 : 设置为建议合的法的回忆招式。 -精灵球 : 弹出球编辑器;合法球的背光是绿色的，非法球是红色的。 -任何标签 : 复制标签到剪切板。

将WC7/WC6/PGF/PCD/PG拖到主界面中转换为PKM。

# 存档

双击SAV选项卡 : 自动检测/重新加载最新保存。

# 盒子

右键单击“盒子”选项卡 : 清除/排序/编辑框内容。选择动作时按住shift键以应用于所有框。

ctrl -右键单击方框槽来查看额外的选项(如合法性检查) control -拖动框槽以复制-覆盖 alt -拖动框槽删除-覆盖

ctrl - alt -点击槽位(克隆/删除)设置一个口袋妖怪到当前盒子中的每个槽位。 ctrl-单击槽位加载槽位到选项卡。 shift -单击槽位设置标签槽位。 alt-单击指定槽位，可删除该槽位上的数据。

双击Box选项卡打开一个新的Box Viewer。一次只允许一个，除非双击时按住shift键。

ctrl - 单击方框箭头 跳转到开始/结束。 Ctrl-左击 Verify checksum以合法性检查保存文件中的所有Pokémon。

// 队伍/对战盒子 双击队伍或战斗盒标签导出Showdown/Smogon sets(团队)到剪贴板。
